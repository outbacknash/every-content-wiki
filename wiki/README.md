# Wiki Content

This directory contains the LLM-maintained knowledge base.

- **Purpose**: To store structured, cross-linked, and synthesized information derived from the `raw/` sources.
- **Directory Structure**:
    - `index.md`: Master catalog of all pages.
    - `log.md`: Append-only activity log.
    - `summaries/`: One summary page per raw source document.
    - `concepts/`: Concept, strategy, and framework pages.
    - `entities/`: People, tools, organizations, and products.
    - `syntheses/`: Cross-cutting analyses and comparisons.
- **Rules**:
    - Content is maintained by the LLM.
    - No orphan pages; everything must be cross-linked.
    - Always follow the schema defined in `CLAUDE.md`.
