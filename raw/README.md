# Raw Sources

This directory contains the immutable source documents for the wiki. 

- **Purpose**: To store the original text, transcripts, and articles from the Every.to subscription.
- **Rules**:
    - Files here are never modified once added.
    - The LLM reads these files to generate and update the structured content in the `wiki/` directory.
    - Supported formats: `.md`, `.txt`, `.pdf` (as text).
